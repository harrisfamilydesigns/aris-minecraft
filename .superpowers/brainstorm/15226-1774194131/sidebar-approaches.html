Sidebar & Controls Layout

How should the sidebar behave across screen sizes?

A

Responsive — Fixed on desktop, drawer on mobile

Desktop keeps the always-visible fixed sidebar pushing content left. Below ~768px it collapses and a "MY BUILDS" button appears to open it as a fullscreen overlay. Recommended — minimal change to the desktop experience.

B

Overlay on all screen sizes

The sidebar becomes a drawer everywhere — no content pushing. Desktop defaults to open, mobile defaults to closed. A toggle button is always visible. Simpler code, but changes the desktop experience more.
