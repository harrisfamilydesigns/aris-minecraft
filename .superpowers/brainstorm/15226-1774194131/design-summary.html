Design Summary

Review before we start building

1. Sidebar → Overlay Drawer

  • Remove margin-right: 180px from #main-content — content fills full width
  • sidebarOpen state in App.jsx, defaults to true on desktop (≥768px), false on mobile
  • Sidebar overlays content — same fixed-right position but higher z-index, no push
  • Semi-transparent backdrop appears behind open drawer; clicking it closes the drawer
  • Smooth slide-in/out transition from the right
  • Sidebar receives isOpen + onClose props; add a close ✕ button inside the drawer header

2. Floating "MY BUILDS" Toggle Button

  • Fixed position, bottom-right corner (above the ground bar)
  • Shows 📦 icon + "MY BUILDS" label, styled to match the game's pixel aesthetic
  • Always visible; toggles the drawer open/closed
  • Rendered in App.jsx alongside Sidebar

3. Controls Move Below Grid / Builder

  • WorldPanel: move .controls div after <WorldGrid /> (currently sits above it)
  • BlockPanel: move .controls div after <BBGrid /> (currently sits above it)
  • No style changes needed — just JSX reorder

Files Changed

  • App.jsx — add sidebarOpen state, floating button, pass props to Sidebar
  • Sidebar.jsx — accept isOpen/onClose, add close button
  • WorldPanel.jsx — move controls below WorldGrid
  • BlockPanel.jsx — move controls below BBGrid
  • index.css — remove margin-right, add drawer/backdrop/floating-btn styles
